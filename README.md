# Trading Experiment: Wiebe vs Claude

90-day trading challenge between human (Wiebe) and AI (Claude).

**Live Dashboard:** https://[jouw-username].github.io/trading-experiment/

## Tech Stack
- Frontend: HTML/CSS/JS (Chart.js)
- Hosting: GitHub Pages
- Database: Supabase
- Live Prices: Supabase Edge Functions

## Setup
1. Fork this repo
2. Enable GitHub Pages (Settings → Pages → Source: `main` branch, `/public` folder)
3. Set up Supabase project
4. Deploy Edge Functions
5. Add Supabase URL + API key to repo secrets

## Local Development
```bash
# Just open public/index.html in browser
open public/index.html
```
